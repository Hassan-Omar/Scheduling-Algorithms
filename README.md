# Scheduling-Algorithms
## content
+  Implments of 5 Schedualing Algorithms
+  RR (Round Robin)
+  FCFS (First Come First Served )
+  SRT (Shortest Remaining Time First ) 
+  SJF (Shortest Job First ) 
+ EDF (Earliest deadLine First )
